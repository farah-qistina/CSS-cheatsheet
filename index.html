HTML Position Reference

Property Value Description
position static default
relative relative to default
absolute
  • element ignored
  • relative to the closest positioned parent element
fixed fixes element to page (regardless of scrolling)
sticky stays in the document flow when the user scrolls but sticks to a specified position when the user scrolls further
float* left floats elements as far left as possible
right floats elements as far right as possible
clear left the left side of the element will not touch any other element within the same containing element
right the right side of the element will not touch any other element within the same containing element
both neither side of the element will not touch any other element within the same containing element
none the element can touch either side

*only works for static and relative elements
